# SmartTasker

Welcome to **SmartTasker** – your intelligent assistant to manage, prioritize, and track tasks effortlessly.

---

## Overview
SmartTasker leverages AI algorithms to help you organize your tasks efficiently. Whether you’re juggling work projects, personal errands, or team collaborations, SmartTasker adapts to your needs and helps you stay focused and productive.

## Features
- AI-powered task prioritization
- Smart reminders and notifications
- Collaborative task sharing
- Seamless integration with popular calendar apps
- Intuitive drag-and-drop task management
- Dark mode and customizable themes

## Tech Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **AI:** TensorFlow.js for local smart prioritization

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/tinaehoward/SmartTasker.git
   ```
2. Navigate into the project folder:
   ```
   cd SmartTasker
   ```
3. Install backend dependencies:
   ```
   npm install
   ```
4. Install frontend dependencies:
   ```
   cd client && npm install
   ```
5. Start the backend server:
   ```
   npm start
   ```
6. In another terminal, start the frontend development server:
   ```
   cd client && npm start
   ```

## Usage
- Open your browser and go to `http://localhost:3000`
- Create an account or sign in with GitHub
- Start adding tasks and watch SmartTasker prioritize and organize your list automatically
- Share tasks with team members for collaborative productivity
